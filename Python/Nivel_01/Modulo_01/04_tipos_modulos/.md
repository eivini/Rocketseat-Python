<div align="flex-start" title="Módulos padrão">
    <h1>
        01 - Módulos padrão: 
    </h1>
    <p>
        São módulos que vêm com a instalação do Python, e você pode usá-los diretamente sem precisar instalar nada. Alguns exemplos de módulos padrão são:
    </p>
    <h3>
        Exemplo:
    </h3>
    <li>
        <b>math:</b> Para funções matemáticas, como <b>sqrt()</b>, <b>sin()</b>, <b>cos()</b>, entre outras.
    </li>
    <li>
        <b>random:</b> Para gerar números aleatórios, como <b>randint()</b> ou <b>choice()</b>.
    </li>
    <li>
        <b>datetime:</b> Para trabalhar com datas e horários.
    </li>
    <li>
        <b>os:</b> Para interagir com o sistema operacional <b>(ex.: manipulação de arquivos e diretórios)</b>.
    </li>
    <li>
        <b>sys:</b> Para interagir com parâmetros e funções do sistema.
    </li>
    <br>
</div>

```python
import math
print(math.sqrt(16))  # Resultado: 4.0
```

<div align="flex-start" title="Módulos personalizados">
    <h1>
        02 - Módulos personalizados: 
    </h1>
    <p>
        São módulos criados por você, ou seja, você pode criar seu próprio módulo e importá-lo para reutilizar código. Isso facilita a organização de grandes programas.
    </p>
    <h3>
        Exemplo:
    </h3>  
    <li>
        Crie um arquivo meu_modulo.py com o código:
    </li>
    <br>
</div>

```python
def saudacao(nome):
    return f"Olá, {nome}!"
```

<div>    
    <li>
        Importe e use em outro script:
    </li>
    <br>
</div>

```python
import meu_modulo
print(meu_modulo.saudacao("eivini"))
```

<div>
    <p>
        São módulos que vêm com a instalação do Python, e você pode usá-los diretamente sem precisar instalar nada. Alguns exemplos de módulos padrão são:
    </p>
    <li>
        <b>requests:</b> Para fazer requisições <b>HTTP</b> de forma simples.
    </li>
    <li>
        <b>numpy:</b> Para operações <b>matemáticas avançadas</b>, especialmente com <b>arrays</b>.
    </li>
    <li>
        <b>pandas:</b> Para <b>análise de dados</b> e <b>manipulação de estruturas de dados</b> como <b>DataFrames</b>.
    </li>
    <li>
        <b>flask</b> ou <b>django:</b> Para desenvolvimento de aplicações web.
    </li>
    <h3>
        Exemplo:
    </h3>
</div>

```python
    import requests
    response = requests.get("https://www.python.org")
    print(response.status_code)  # Exemplo de uso do módulo requests
```


<div align="flex-start" title="Pacotes">
    <h1>
        Pacotes: 
    </h1>
    <p>
        Pacotes são uma coleção de módulos agrupados em um diretório, geralmente com um arquivo __init__.py. Isso permite organizar módulos de forma hierárquica.
    </p>
    <h3>
        Exemplo:
    </h3>
    <li>
        Um pacote chamado meu_pacote pode ter a estrutura:
    </li>
    <br>
</div>

```python
meu_pacote/
    __init__.py
    modulo1.py
    modulo2.py
```

<div>
    <li>
        E você pode importar os módulos assim:
    </li>
    <br>
</div>

```python
from meu_pacote import modulo1
```
