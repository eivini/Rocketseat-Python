##  01 - Módulos padrão: 
    São módulos que vêm com a instalação do Python, e você pode usá-los diretamente sem precisar instalar nada. Alguns exemplos de módulos padrão são:

    • math: Para funções matemáticas, como sqrt(), sin(), cos(), entre outras.

    • random: Para gerar números aleatórios, como randint() ou choice().

    • datetime: Para trabalhar com datas e horários.

    • os: Para interagir com o sistema operacional (ex.: manipulação de arquivos e diretórios).

    • sys: Para interagir com parâmetros e funções do sistema.

    Exemplo de uso:

    import math
    print(math.sqrt(16))  # Resultado: 4.0

##  02 - Módulos personalizados: 
    São módulos criados por você, ou seja, você pode criar seu próprio módulo e importá-lo para reutilizar código. Isso facilita a organização de grandes programas.

    Exemplo:

    • Crie um arquivo meu_modulo.py com o código:

    def saudacao(nome):
        return f"Olá, {nome}!"

    • Importe e use em outro script:

    import meu_modulo
    print(meu_modulo.saudacao("João"))

##  03 - Módulos de terceiros: 
    São módulos que você instala utilizando o gerenciador de pacotes pip. São desenvolvidos por terceiros e fornecem funcionalidades adicionais para Python. Exemplos incluem:

    • requests: Para fazer requisições HTTP de forma simples.

    • numpy: Para operações matemáticas avançadas, especialmente com arrays.

    • pandas: Para análise de dados e manipulação de estruturas de dados como DataFrames.

    • flask ou • django: Para desenvolvimento de aplicações web.

    Exemplo de uso:

    import requests
    response = requests.get("https://www.python.org")
    print(response.status_code)  # Exemplo de uso do módulo requests

##  Pacotes: 
    Pacotes são uma coleção de módulos agrupados em um diretório, geralmente com um arquivo __init__.py. Isso permite organizar módulos de forma hierárquica.

    Exemplo:

    Um pacote chamado meu_pacote pode ter a estrutura:

    meu_pacote/
        __init__.py
        modulo1.py
        modulo2.py

    • E você pode importar os módulos assim:

    from meu_pacote import modulo1
