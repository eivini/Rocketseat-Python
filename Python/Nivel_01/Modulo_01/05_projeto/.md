##  Primiro Projeto Rocketseat:
    Um gerenciador de lista de tarefas em Python, que permite ao usuário adicionar, visualizar, atualizar, completar e deletar tarefas de uma lista. Ele utiliza um loop while para exibir um menu e realizar as ações conforme a escolha do usuário. Aqui está um resumo de cada parte:

>   Função adicionar_tarefa:

    Adiciona uma nova tarefa à lista de tarefas. Cada tarefa é representada por um dicionário com o nome da tarefa e um status (completada, inicialmente False).

>   Função ver_tarefas:

    Exibe todas as tarefas da lista com um status de "completada" marcado como ✓ (se a tarefa foi completada) ou um espaço em branco (se não foi completada).

>   Função atualizar_nome_tarefas:

    Permite ao usuário atualizar o nome de uma tarefa, dado seu índice. O índice fornecido pelo usuário é ajustado para o índice correto da lista (que começa em 0).

>   Função completar_tarefa:

    Marca uma tarefa como completada, alterando seu status para True.

>   Função deletar_tarefas_completadas:

    Remove todas as tarefas que já foram completadas da lista.

### Loop principal (while True):

    Exibe um menu com as opções: adicionar, ver, atualizar, completar e deletar tarefas, ou sair do programa.

    O usuário escolhe uma opção e o programa executa a função correspondente. O loop continua até que o usuário escolha a opção de sair (6).

##  Como o programa funciona:
    O programa começa com uma lista vazia de tarefas.

    O menu é exibido continuamente até que o usuário escolha a opção para sair.

    As ações disponíveis são:

        1: Adicionar uma tarefa.

        2: Ver as tarefas.

        3: Atualizar o nome de uma tarefa.

        4: Completar uma tarefa.

        5: Deletar tarefas completadas.

        6: Sair do programa.

##  Exemplo de interação:
    O usuário escolhe "1" para adicionar uma tarefa.

    O usuário digita o nome da tarefa e ela é adicionada à lista.

    O usuário escolhe "2" para ver as tarefas, e o programa mostra todas as tarefas com seus status (completada ou não).

    O usuário pode então atualizar o nome de uma tarefa, marcar uma tarefa como completada, ou deletar tarefas já completadas.

>   O programa continuará rodando até o usuário escolher a opção para sair.
