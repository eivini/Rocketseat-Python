O que é um Decorador?
Um decorador em Python é uma função que modifica o comportamento de outra função, método ou classe. Ele "decorar" uma função, ou seja, adiciona funcionalidades antes ou depois da execução dessa função, sem modificar o código original.

Em termos simples, um decorador é uma função que recebe outra função como argumento e retorna uma nova função que, normalmente, executa algo antes ou depois da função original.

Como funciona um Decorador?
Em Python, você pode usar a sintaxe @decorator antes da definição de uma função para aplicar o decorador. O que acontece por trás dos panos é que o decorador substitui a função original pela função decorada.

Exemplo básico de um Decorador:
Vamos criar um decorador simples que imprime uma mensagem antes de chamar a função decorada.

python
Copiar
# Definindo o decorador
def meu_decorador(func):
    def wrapper():
        print("Antes da execução da função.")
        func()  # Chama a função original
        print("Depois da execução da função.")
    return wrapper

# Aplicando o decorador à função
@meu_decorador
def minha_funcao():
    print("Executando minha função.")

# Chamando a função decorada
minha_funcao()

Explicação:

O decorador meu_decorador recebe a função minha_funcao como argumento e cria uma nova função wrapper que envolve a execução de minha_funcao.

Quando chamamos minha_funcao(), o que realmente está acontecendo é a chamada de wrapper(), que executa o código extra (mensagens de antes e depois) e, em seguida, chama a função original.

Decoradores Comuns
Decoradores comuns em Python são decoradores que realizam tarefas frequentes e úteis, como verificação de acesso, medição de tempo de execução, log de chamadas de funções, etc. Vou te mostrar alguns exemplos de decoradores comuns que são bastante utilizados.

1. Decorador para medir o tempo de execução:
Este decorador mede quanto tempo uma função leva para ser executada.

python
Copiar
import time

def medir_tempo(func):
    def wrapper(*args, **kwargs):
        inicio = time.time()
        resultado = func(*args, **kwargs)  # Executa a função
        fim = time.time()
        print(f"Tempo de execução: {fim - inicio} segundos.")
        return resultado
    return wrapper

@medir_tempo
def exemplo_lento():
    time.sleep(2)  # Simula uma operação demorada

exemplo_lento()
Saída:

Copiar
Tempo de execução: 2.0001235 segundos.
Explicação:

O decorador medir_tempo calcula o tempo antes e depois da execução da função original.

A função exemplo_lento foi decorada com @medir_tempo, então quando ela é chamada, o tempo de execução é medido.

2. Decorador para verificar permissão de acesso:
Vamos criar um decorador simples que verifica se o usuário tem permissão para acessar determinada função.

python
Copiar
def verificar_permissao(func):
    def wrapper(usuario, *args, **kwargs):
        if usuario == "admin":
            return func(usuario, *args, **kwargs)
        else:
            print(f"Usuário {usuario} não tem permissão!")
            return None
    return wrapper

@verificar_permissao
def acessar_area_restrita(usuario):
    print(f"Bem-vindo à área restrita, {usuario}!")

acessar_area_restrita("admin")  # Permissão concedida
acessar_area_restrita("usuario")  # Permissão negada

pgsql
Copiar
Bem-vindo à área restrita, admin!
Usuário usuario não tem permissão!
Explicação:

O decorador verificar_permissao verifica se o usuário tem o nome "admin". Se sim, a função original é executada; caso contrário, uma mensagem de erro é exibida.

A função acessar_area_restrita foi decorada com @verificar_permissao, então antes de executar, o decorador verifica as permissões.
