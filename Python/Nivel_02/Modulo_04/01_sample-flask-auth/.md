#   Para que serve o Banco de dados?
    Projetado para armazenamento, recuperação e gerenciamento de informação


#   Memória volátil (RAM):
    Memória ram é uma peça(hardware) que serve para armazenar informações igual o disco rígido(HD), porém ela tem uma velocidade muito superior,
    mas ela não perciste dados(ela não salva os dados)


#   O que é o Banco de dados?
    Uma ferramenta para percistir a informação(não perder os dados/armazenamento de dados) 


#   FlaskSQLAlchemy:    (Biblioteca) É uma extensão pro flask que suporta o SQLAlchemy na sua aplicação
    https://flask-sqlalchemy.readthedocs.io/en/stable/


#   SQLAlchemy é um ORM:
    • O que é um ORM?
        - Significado: Object Relational Mapper
        - É um mapeador de objetos relacional

    • Qual a vantagem de utilizar um ORM?
        - Se em algum momento eu precisar trocar de banco, é possível fazer de uma forma mais fácil,
        atráves do ORM ele faz a abstração, então não é necessário reescrever os códigos, pois a inserção
        vai utilizar o mesmo método

    • Qual a desvantagem?
        - Se torna dependente do ORM, caso queira trocar de ORM vai ser bem complicado (principalmente em grandes empresas)
