#   O que é o Pytest?
    um framework do python que ajuda a garantir a qualidade dos programas atraves de testes automatizados


#   Como instalar?
    pip3 install -r requirements.txt


#   Teste pelo pytest pelo cmd/terminal
    pytest teste.py -v
    pytest teste.py -verbose


#   Para testar é necessário executar o app.py e depois o testes.py
