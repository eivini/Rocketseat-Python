##  • Criar novas tarefas:

#   [Post]
    POST > {{baseUrl}}/tasks
    {
        "title": "Estudar Python",
        "description": "API"
    }
    click Send > Response = Teste
    no terminal vai mostrar as informações do request e o valor retornado exemplo: [200]

+   outra tarefa:
    POST > {{baseUrl}}/tasks
    {
        "title": "Estudar Java",
        "description": "API"
    }

#   [GET]
    GET > {{baseUrl}}/tasks
    {
        "tasks": [
            {
                "completed": false,
                "description": "",
                "id": 1,
                "title": "Estudar Matematica"
            }
        ]
        "total_tasks": 0
    }

#   [PUT]
    PUT > {{baseUrl}}/tasks
    {
        "title": "Estudar Flask",
        "description": "estudar CRUDs",
        "completed": false
    }


#   Ir no + como se fosse abrir uma nova aba no navegador, passar o URL e o parâmetro
    Exemplo: (Caso seja um parâmetro existente = 200)
        {{baseUrl}}/tasks/1
        e ele vai retornar a tasks específica: 
        {
            "completed": false,
            "description": "",
            "id": 1,
            "title": "Estudar APIs"
        }
    Exemplo 2: (Caso não seja um parâmetro existente = 404)
        {{baseUrl}}/tasks/10
        {
            "message": "Não foi possível encontrar a atividade"
        }


#   OBS: Caso queira salvar sua requisição, é possível:
        só clicar em: Save > Selecionar a collection or folder > Colocar o nome > Salvar
        e ela fica visivelmente disponível
