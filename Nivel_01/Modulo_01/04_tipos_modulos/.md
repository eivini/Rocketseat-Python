
<div align="flex-start" title="Módulos padrão">
    <h1>01 - Módulos padrão:</h1>
    <p>
        São módulos que vêm com a instalação do Python, e você pode usá-los diretamente sem precisar instalar nada.
        Alguns exemplos de módulos padrão são:
    </p>
    <h3>Exemplos:</h3>
    <ul>
        <li><b>math:</b> Para funções matemáticas, como <b>sqrt()</b>, <b>sin()</b>, <b>cos()</b>, entre outras.</li>
        <li><b>random:</b> Para gerar números aleatórios, como <b>randint()</b> ou <b>choice()</b>.</li>
        <li><b>datetime:</b> Para trabalhar com datas e horários.</li>
        <li><b>os:</b> Para interagir com o sistema operacional (ex.: manipulação de arquivos e diretórios).</li>
        <li><b>sys:</b> Para interagir com parâmetros e funções do sistema.</li>
    </ul>
</div>

```python
import math
print(math.sqrt(16))  # Resultado: 4.0
```

<div align="flex-start" title="Módulos personalizados">
    <h1>02 - Módulos personalizados:</h1>
    <p>
        São módulos criados por você, ou seja, você pode criar seu próprio módulo e importá-lo para reutilizar código.
        Isso facilita a organização de grandes programas.
    </p>
    <h3>Exemplo:</h3>
    <ul>
        <li>Crie um arquivo <code>meu_modulo.py</code> com o código:</li>
    </ul>
</div>

```python
def saudacao(nome):
    return f"Olá, {nome}!"
```

<div align="flex-start" title="Complemento">
    <ul>
        <li>Importe e use em outro script:</li>
    </ul>
</div>

```python
import meu_modulo
print(meu_modulo.saudacao("eivini"))
```

<div align="flex-start" title="Módulos de terceiros">
    <h1>03 - Módulos de terceiros:</h1>
    <p>
        São módulos que você instala com o <code>pip</code>. Eles ampliam as funcionalidades do Python com bibliotecas criadas por terceiros.
    </p>
    <h3>Instalação:</h3>
</div>

```bash
pip install requests==2.31.0
```

<div align="flex-start" title="Exemplos de módulos de terceiros">
    <h3>Exemplos:</h3>
    <ul>
        <li><b>requests:</b> Para fazer requisições <b>HTTP</b> de forma simples.</li>
        <li><b>numpy:</b> Para operações matemáticas avançadas, especialmente com arrays.</li>
        <li><b>pandas:</b> Para análise de dados e manipulação de <b>DataFrames</b>.</li>
        <li><b>flask</b> ou <b>django:</b> Para desenvolvimento de aplicações web.</li>
    </ul>
</div>

```python
import requests
response = requests.get("https://www.python.org")
print(response.status_code)
```

<div align="flex-start" title="Pacotes">
    <h1>04 - Pacotes:</h1>
    <p>
        Pacotes são uma coleção de módulos agrupados em um diretório com um arquivo <code>__init__.py</code>.
        Isso permite organizar módulos de forma hierárquica.
    </p>
    <h3>Exemplo de estrutura:</h3>
</div>

```plaintext
meu_pacote/
├── __init__.py
├── modulo1.py
└── modulo2.py
```

<div align="flex-start" title="Importando pacotes">
    <ul>
        <li>E você pode importar os módulos assim:</li>
    </ul>
</div>

```python
from meu_pacote import modulo1
```
