<div align="flex-start" title="for">
    <h1>
        01 - for: 
    </h1>
    <p>
        O laço for é usado para iterar sobre uma sequência (como uma lista, tupla, string ou intervalo de números). Ele repete um bloco de código para cada item na sequência.
    </p>
    <h3>
        Exemplo:
    </h3>
</div>

```python
for i in range(5):  # Itera de 0 a 4
    print(i)

```

<div align="flex-start" title="Complemento">
    <p>
        Neste exemplo, o <b>for</b> vai imprimir os números de 0 a 4.
    </p>
</div>

<br>
<div align="flex-start" title="while">
    <h1>
        02 - while:
    </h1>
    <p>
        O laço while repete um bloco de código enquanto uma condição for verdadeira. Ele continua executando até que a condição se torne falsa.
    </p>
    <h3>
        Exemplo:
    </h3>
</div>

```python
i = 0
while i < 5:  # Repete enquanto i for < 5
    print(i)
    i += 1
```

<div title="Complemento">
    <p>
        Neste caso, o <b>while</b> vai imprimir os números de 0 a 4, e a execução termina quando a condição i < 5 não for mais verdadeira.
    </p>
</div>
